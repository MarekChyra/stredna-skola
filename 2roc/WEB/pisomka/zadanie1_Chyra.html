Kaskádové štýly

Pomocou jazyka CSS vytvárame vzhľad stránky resp. formátovanie stránky. Štýly umožnili oddeliť štruktúru HTML od vzhľadu.
Zápis vzhľadu stránky sa robí pomocou množiny vlastností a ich hodnôt. Túto množinu hodnôt potom aplikujeme na jeden alebo viacero prvkov na stránke.

Štýly môžeme zapisovať tromi rôznymi spôsobmi:

- zápis do externého súboru (externý zápis) – k stránke pripojíme súbor s príponou css
- zápis do hlavičky webstránky (interný zápis) - do hlavičky HTML súboru umiestnime párový tag
- priamy zápis štýlu do HTML značky (inline zápis) – zápis štýlov priamo do HTML značky
