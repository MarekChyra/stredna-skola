Počitačové siete je súhrnné označenie počítačov, pomocou ktorých je realizované prepojenie a výmena dát medzi počítačmi. Umožňuje používateľom komunikáciu podľa zadaných pravidiel. Najčastejším dôvodom pripojenia k sieti je zdieľanie informácií a technických zariadení.

Rozdelenie počitačových sietí podľa topológie:

  • Zbernicová topológia siete
  • Hviezdicová topológia siete
  • Prstencová topológia siete

Rozdelenie počitačových sietí podľa veľkosti

  • Personal area network (PAN)
  • Local area network (LAN)
  • Metropolitan area network (MAN)
  • Wide area network (WAN)

ISO/OSI model

  1. Aplikačná vrstva
  2. Prezenčná vrstva
  3. Relačná vrstva
  4. Transportná vrstva
  5. Sieťová vrstva
  6. Spojová vrstva
  7. Fyzická vrstva
